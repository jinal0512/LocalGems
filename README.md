# LocalGems

LocalGems is a Django-based web application that helps users discover and promote local businesses.

## 🚀 Features
- User authentication (Login/Register)
- Add and manage local businesses
- Search functionality
- Category-based filtering

## 🛠 Tech Stack
- Python
- Django
- PostgreSQL
- HTML, CSS

## 🗄 Database
This project uses PostgreSQL for efficient data storage and management.

## ▶️ How to Run
1. Clone the repository
2. Install dependencies:
   pip install -r requirements.txt
3. Setup PostgreSQL database
4. Run migrations:
   python manage.py migrate
5. Run the server:
   python manage.py runserver
